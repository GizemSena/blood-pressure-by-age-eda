🩺 Blood Pressure by Age — Exploratory Data Analysis
📌 Project Overview

This project performs an Exploratory Data Analysis (EDA) on a global blood pressure dataset to analyze how blood pressure varies across different age groups and demographic factors.

Blood pressure is one of the most important indicators of cardiovascular health, and hypertension affects over 1.4 billion adults worldwide, making it a major global health concern.

The goal of this analysis is to explore patterns in blood pressure measurements and understand how factors such as age and gender relate to changes in blood pressure levels.

📂 Dataset

Source: Kaggle
Dataset: Blood Pressure by Age – Global Dataset

The dataset contains synthetic health records including:

Patient ID
Age
Gender
Systolic Blood Pressure
Diastolic Blood Pressure
Additional health-related attributes

The dataset is suitable for:

Exploratory Data Analysis (EDA)
Health data visualization
Hypertension trend analysis
Educational machine learning projects

Blood pressure comparison between genders
Boxplot visualization
6️⃣ Correlation Analysis
Correlation matrix between numeric health variables


📊 Key Insights

Important findings from the analysis include:

Blood pressure tends to increase gradually with age.
Systolic and diastolic blood pressure values show strong correlation, which is expected as they are components of the same physiological measurement.
Some variability exists between demographic groups such as gender.
Age appears to be one of the most influential factors affecting blood pressure levels.
